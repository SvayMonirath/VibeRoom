**LUNA** is a web-based music platform that allows users to create and join real-time shared music rooms. Users can listen to songs together, chat, and enjoy a synchronized music experience with friends or the public.

---

## Features

- **Real-Time Shared Music Rooms**: Create or join rooms to listen to music simultaneously.
- **Room Management**: Host controls for adding, removing, and playing songs.
- **User Authentication**: Secure login and registration system.
- **Music Player UI**: Stylish, responsive, and interactive music player.
- **Chat & Interaction**: Communicate with other users in the same room.
